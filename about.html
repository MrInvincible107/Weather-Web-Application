This is a weather app designed by Saksham Budhiraja BSc Computer Science Student. In this app, we get default weather details about Delhi,India. It also gives details about some more cities in brief such as Shanghai, London, New York, etcetra. If you want to search for a custom city, you get an option in the search bar where you can type the city name and press search in order to search for your desired city and the results will be shown in the main three cards at the top of common cities. This project uses HTML, CSS and Javascript to develop the structure and logics. This use Weather by API-Ninjas as its Api. Other than this there is no API or library that is used externally.
